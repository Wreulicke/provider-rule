# provider-rule

# Installation

add repository and dependency.
also see [jitpack](https://jitpack.io/#Wreulicke/provider-rule/0.0.2).

* Example for gradle

```groovy
  repositories {
    maven { url 'https://jitpack.io' }
  }

  dependencies {
    compile 'com.github.Wreulicke:provider-rule:0.0.2'
  }
```
